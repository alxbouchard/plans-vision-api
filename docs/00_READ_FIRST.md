# MCAF Execution Pack
Date: 2025-12-25

## But
Ce pack évite la dérive et élimine les discussions circulaires.
Il définit la façon officielle de terminer l’API plans-vision-api selon MCAF.

## Règle d’or
Le code n’invente rien.
Le guide est la seule source d’intelligence.
Les moteurs exécutent, ils ne décident pas.

## Comment utiliser ce pack
1. Lire 01_MCAF_RULES.md
2. Lire 02_PROJECT_MAP.md
3. Lire 06_WHAT_IS_FROZEN.md
4. Suivre 04_DEV_CHECKLIST.md dans l’ordre
5. Appliquer 05_TEST_GATES_MASTER.md avant chaque merge

## Définition de “Terminé”
Une phase est terminée seulement si:
- ses gates passent
- les décisions associées sont écrites
- les composants gelés ne sont pas modifiés
